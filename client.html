2How to get your Gmail App Password

Takes 2 minutes. Just follow these clicks.

  1. Open this link: myaccount.google.com/security
  2. Find "2-Step Verification" and turn it ON (skip if it's already on).
  3. Now open this link: myaccount.google.com/apppasswords
  4. Type any name, like "Job Alerts", then click Create.
  5. Google will show you a code with 16 letters. Copy this code — this is your App Password.
Is this safe? Yes. This code only lets the system check for new job emails — nothing else. It cannot see your other emails, cannot post anything, cannot access your bank or personal info. You can cancel it anytime from the same page.
